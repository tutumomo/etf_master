# 06-02-PLAN: 優化仲裁邏輯 (generate_decision_consensus.py)

## Objective
優化仲裁邏輯，增加「策略對齊檢核」，並在結論輸出中明確標註建議是否對齊策略。

## Context
目前的 `generate_decision_consensus.py` 僅綜合各方建議，但未深入檢查建議標的是否符合當前策略（如：收益策略下若建議標的分類非 income，則降低評分）。

## Tasks
- [ ] 1. 實作「策略對齊檢核」邏輯：檢查建議標的是否符合當前策略 @auto
- [ ] 2. 修改仲裁結論輸出格式：明確標註建議是否對齊策略 @auto
- [ ] 3. 更新仲裁權重與評分規則，將策略對齊度納入最終評分 @auto
- [ ] 4. 驗證仲裁結果的準確性與對齊標註 @auto

## Success Criteria
- `generate_decision_consensus.py` 能夠識別建議標的與當前策略的衝突。
- `decision_consensus.json` 中的輸出包含明確的對齊狀態標籤（如：`aligned: true/false`）。
- 仲裁評分能夠反應策略對齊度的權重影響。
