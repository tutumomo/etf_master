```json
{
  "strategy_alignment_rate": 0.0,      // float, percent 0-100, records where strategy_alignment==True / total with that field
  "confidence_distribution": {
    "high": 0,                          // confidence >= 0.7
    "medium": 0,                        // 0.4 <= confidence < 0.7

  },
  "interception_rate": 0.0,            // float, percent: pre_flight_intercepted==True / total records

  "last_updated": "ISO8601 string"
}
```

Confidence bucketing: if `confidence` key absent on a record, skip it for distribution counts (do not error).
`strategy_alignment_rate` denominator is count of records that have `strategy_alignment` key (not total).
`interception_rate` denominator is total records.
