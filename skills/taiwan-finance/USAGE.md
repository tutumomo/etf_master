# Taiwan Finance Suite 使用說明 (USAGE.md)

`taiwan-finance` 是一個為專業金融從業者設計的綜合性技能套件，整合了投資銀行 (IB)、個股研究 (ER)、私募股權 (PE)、財富管理 (WM) 及核心財務分析 (FA) 的專業框架與在地化數據檢索邏輯。

## 如何調用專業框架 (Slash Commands)

當您與 Agent 對話時，可以使用以下斜槓指令或關鍵字來啟動特定的分析流程：

### 1. 公司估值與財務建模 (`/comps`, `/dcf`)
調用 `financial-analysis.md` 中的估值邏輯，針對台灣市場進行深度分析。
- **`/comps [代碼]`**: 執行可比公司分析。Agent 會自動檢索台股與全球同業的 P/E, P/B, EV/EBITDA 倍數進行橫向比較。
- **`/dcf [代碼]`**: 執行現金流量折現法建模。Agent 會套用台灣市場參數 (如台債 10Y 作為 Rf) 計算企業內含價值。

### 2. 個股研究與財報掃描 (`/earnings`)
調用 `equity-research.md` 中的研究邏輯，專注於基本面轉折點。
- **`/earnings [代碼]`**: 檢索最新法說會資訊、月營收動能 (YoY/MoM) 與三表關鍵數據。
- **關鍵字：[法說會摘要]**: 要求 Agent 總結公開資訊觀測站 (MOPS) 的最新報告。

### 3. 投行與私募文件撰寫 (`/memo`, `/teaser`)
調用 `investment-banking.md` 與 `private-equity.md` 中的專業格式。
- **`/memo [項目名稱]`**: 撰寫標準的投委會 (IC) 投資建議備忘錄。
- **`/teaser [公司名稱]`**: 生成專業的併購案初步簡介 (Teaser)，包含亮點摘要與核心財務數據。

### 4. 財富管理與資產配置 (`/plan`)
調用 `wealth-management.md` 中的規劃框架。
- **`/plan [客戶情境]`**: 針對台灣稅務環境 (如：所得稅、贈與稅、海外所得) 執行資產配置與退休規劃建議。

## 核心分析模組 (Module Directory)

| 模組 | 適用領域 | 核心功能 |
|---|---|---|
| **FA (財務分析)** | 估值、三表、LBO | Comps/DCF 建模、月營收動能分析 |
| **ER (個股研究)** | 法說會、催化劑、月營收 | 業績轉折點偵測、研究報告摘要 |
| **IB (投資銀行)** | Pitch Deck、併購、CIM | 融資併購模型、專業交易文件製作 |
| **PE (私募股權)** | DD、案源、單元經濟學 | 盡職調查清單、產業結構分析 |
| **WM (財富管理)** | 資產配置、財務規劃 | 稅務優化、退休儲蓄規劃 |

## 數據獲取規範

本技能優先從以下來源獲取「真相」資料：
1. **公開資訊觀測站 (MOPS)**: 獲取官方財報與月營收。
2. **台灣證券交易所 (TWSE/TPEx)**: 獲取即時報價與三大法人籌碼。
3. **Yahoo Finance**: 獲取歷史價格與全球對標數據。
