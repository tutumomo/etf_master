# etf_core

這是 ETF_TW 內部核心模組的說明文件。

## 目前定位
- 所屬技能：Hermes 版 `ETF_TW`
- 技能根目錄：`~/.hermes/profiles/etf_master/skills/ETF_TW`
- 不再使用任何舊的工作目錄作為主開發路徑

## 安裝
```bash
cd ~/.hermes/profiles/etf_master/skills/ETF_TW
python3 -m venv .venv
source .venv/bin/activate
pip install -r scripts/etf_core/requirements.txt
```
