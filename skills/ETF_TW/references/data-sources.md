# 資料來源與更新規則

## 資料來源

### 1. 股價資料
- **來源**：Yahoo Finance (`yfinance`)
- **更新頻率**：盤中即時（可能有延遲），盤後收盤價
- **涵蓋標的**：台股（.TW 後綴）、美股、其他國際市場

**限制**：
- 台灣股市：9:00-13:30（週一至週五）
- 可能有 15 分鐘延遲
- 非交易時間為最後收盤價
- 不保證資料完整性

### 2. 技術指標
- **計算方式**：自行計算（非外部 API）
- **指標類型**：MA5/10/20、RSI(14)、MACD
- **資料基礎**：歷史股價（Yahoo Finance）

**限制**：
- 依賴歷史資料品質
- 短期指標可能波動大
- 不保證訊號準確性

### 3. 新聞情報
- **來源**（規劃中）：Yahoo 財經、PTT 股版、CNBC
- **情緒評分**：NLP 分析（未來功能）

**限制**：
- 目前為 placeholder
- 未實作爬蟲時不提供新聞

### 4. ETF 基本資料
- **來源**：Yahoo Finance info
- **欄位**：名稱、費用率、配息率、追蹤指數、規模

**限制**：
- 部分欄位可能缺失
- 費用率可能非最新
- 需交叉驗證

---

## 資料更新規則

### 即時資料（盤中）
- 每次查詢時重新取得
- 不緩存（避免過時）
- 明確標註時間戳

### 歷史資料
- 可緩存短期（如：5 分鐘）
- 長期資料需重新抓取
- 明確標註資料日期

### 基本資料
- 每日更新一次（規劃中）
- 重大變動時手動更新
- 記錄最後更新時間

---

## 誠實原則

### 不宣稱即時
若資料未經當次取得，必須明確標註：
```
⚠️ 資料提示：
- 股價為 2026-03-23 收盤價
- 非即時報價
- 實際成交價可能不同
```

### 不保證準確
所有分析僅供參考：
```
⚠️ 免責聲明：
- 技術指標僅供參考
- 不構成投資建議
- 過去表現不代表未來
```

### 不隱藏限制
若資料有缺失或限制，明確說明：
```
⚠️ 資料限制：
- 該 ETF 費用率資料缺失
- 無法取得最新配息紀錄
- 建議交叉驗證其他來源
```

## 資料品質檢查

### 每次查詢時檢查
- [ ] 資料是否為空
- [ ] 價格是否合理（非負數、非異常值）
- [ ] 時間戳是否新鮮
- [ ] 欄位是否完整

### 發現異常時
- 標註資料可能不準確
- 建議使用者交叉驗證
- 不提供基於異常資料的執行建議
